# Encryption and decryption of messages using symmetric and asymmetric methods

This project is an example of how to create a VSC script that test encryption and decryption of messages using symmetric and asymmetric methods. Lines 3-21 test the symmetric method with one key, while lines 28-45 test the asymmetic method using two separate linked keys.

# How to install this VSC project on GitHub
Download the VSC link in GitHub
In VSC go to file, then open, then open the file where the VSC file is saved
Run the code and you should see both messages using both methods
